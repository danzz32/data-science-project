Introdução

Este notebook tem como objetivo documentar o perfil de qualidade dos dados brutos de acidentes da PRF (camada raw/). O diagnóstico guiará as decisões de tratamento que serão implementadas no script transform.py. A análise está dividida nas quatro dimensões de Batini et al. (2009): Completude, Consistência, Acurácia e Temporalidade.

Objetivo: Identificar campos com valores ausentes (nulos), a proporção de falta e se existe algum padrão.

import pandas as pd

# Calcula a percentagem de nulos por coluna
nulos = df.isnull().sum()
nulos_pct = (nulos / len(df)) * 100

# Filtra apenas colunas que possuem nulos e ordena de forma decrescente
df_nulos = pd.DataFrame({
    'Total_Nulos': nulos,
    'Percentagem (%)': nulos_pct
}).sort_values(by='Total_Nulos', ascending=False)

# Criamos uma cópia formatada para exibição visual limpa
df_nulos_view = df_nulos.copy()
df_nulos_view['Percentagem (%)'] = df_nulos_view['Percentagem (%)'].map('{:.2f}%'.format)

print("📊 Tabela Geral de Campos Ausentes:")
display(df_nulos_view[df_nulos['Total_Nulos'] > 0])

# ==========================================
# PARTE 2: Investigação de Padrões (O Requisito do PDF!)
# ==========================================
# Vamos escolher a coluna com maior número de nulos para investigar o padrão.
# Substitua 'municipio' ou a coluna crítica que encontrar no seu print abaixo:
coluna_critica = df_nulos[df_nulos['Total_Nulos'] > 0].index[0] if any(nulos > 0) else None

if coluna_critica:
    print(f"\n🔍 Investigando padrões de ausência para a coluna: '{coluna_critica}'")
    
    # Criamos uma coluna booleana temporária indicando se o dado é nulo
    df['is_null_critico'] = df[coluna_critica].isnull()
    
    # A. Padrão Temporal: Os nulos estão concentrados em algum ano/período específico?
    # Extrai o ano se a coluna data_inversa já tiver sido convertida no notebook
    if pd.api.types.is_datetime64_any_dtype(df['data_inversa']):
        df['ano_analise'] = df['data_inversa'].dt.year
    else:
        df['ano_analise'] = pd.to_datetime(df['data_inversa'], errors='coerce').dt.year
        
    padrao_temporal = pd.crosstab(df['ano_analise'], df['is_null_critico'], normalize='index') * 100
    print("\n📅 Distribuição de Ausências por Ano (%):")
    display(padrao_temporal.round(2))
    
    # B. Padrão Categórico: Os nulos se concentram em alguma categoria (Ex: UF ou Classificação)?
    if 'uf' in df.columns:
        padrao_categorico = pd.crosstab(df['uf'], df['is_null_critico'], normalize='index') * 100
        print("\n📍 Distribuição de Ausências por UF (%):")
        display(padrao_categorico.round(2))
        
    # Limpeza de colunas temporárias de diagnóstico
    df.drop(columns=['is_null_critico', 'ano_analise'], inplace=True, errors='ignore')
else:
    print("\n✨ Excelente! Nenhuma coluna possui valores nulos nativos no DataFrame.")

2. Dimensão: Consistência

Objetivo: Verificar contradições internas na base de dados.

import pandas as pd

print(“🛡️ Iniciando Testes de Consistência Interna…”)

1. Validação de Não-Negativos (Seu teste original expandido para todas as métricas)

colunas_numericas = [‘mortos’, ‘feridos_leves’, ‘feridos_graves’, ‘ilesos’, ‘ignorados’, ‘veiculos’, ‘pessoas’] if all(c in df.columns for c in colunas_numericas):

# Check A: Valores negativos onde não deveria haver
condicao_negativo = False
for col in colunas_numericas:
    condicao_negativo |= (df[col] < 0)
negativos = df[condicao_negativo]
print(f"❌ Regra 1: Registros com valores numéricos negativos: {len(negativos)}")

# Check B: Vítimas sem veículos (Seu teste original)
sem_veiculo = df[(df['veiculos'] == 0) & ((df['mortos'] > 0) | (df['feridos_leves'] > 0) | (df['feridos_graves'] > 0))]
print(f"❌ Regra 2: Acidentes com vítimas mas zero veículos envolvidos: {len(sem_veiculo)}")

# Check C: Consistência na Matemática de Vítimas (Soma dos componentes vs Total)
# feridos_leves + feridos_graves deve ser igual a feridos
if 'feridos' in df.columns:
    soma_feridos_incorreta = df[df['feridos_leves'] + df['feridos_graves'] != df['feridos']]
    print(f"❌ Regra 3: Incoerência na soma de feridos (Leves + Graves != Total): {len(soma_feridos_incorreta)}")

# Total de pessoas deve ser a soma de todas as categorias de estado físico
soma_total_incorreta = df[
    (df['mortos'] + df['feridos_leves'] + df['feridos_graves'] + df['ilesos'] + df['ignorados']) != df['pessoas']
]
print(f"❌ Regra 4: Incoerência na soma total de pessoas (Categorias != Campo Pessoas): {len(soma_total_incorreta)}")

# Check D: Consistência Textual vs Numérica (Classificação do Acidente)
if 'classificacao_acidente' in df.columns:
    # Padronização temporária para validação justa
    classe_clean = df['classificacao_acidente'].astype(str).str.lower()
    
    # Contradição: Diz que é com vítimas mortas, mas contagem de mortos é zero
    mortos_contraditorios = df[(classe_clean.str.contains('morta')) & (df['mortos'] == 0)]
    # Contradição: Diz que é sem vítimas, mas tem contagem de feridos ou mortos
    sem_vitima_contraditorio = df[(classe_clean.str.contains('sem vít')) & ((df['mortos'] > 0) | (df['feridos_leves'] > 0) | (df['feridos_graves'] > 0))]
    
    print(f"❌ Regra 5: Contradição em 'Com Vítimas Mortas' mas com zero mortos no contador: {len(mortos_contraditorios)}")
    print(f"❌ Regra 6: Contradição em 'Sem Vítimas' mas registrando feridos/mortos: {len(sem_vitima_contraditorio)}")

# Check E: Consistência Geográfica básica (Coordenadas zeradas/nulas fora do padrão)
if 'latitude' in df.columns and 'longitude' in df.columns:
    # No Brasil, latitudes são negativas. Valores exatamente 0 são erros clássicos de GPS
    coordenadas_invalidas = df[(df['latitude'] == 0) | (df['longitude'] == 0)]
    print(f"❌ Regra 7: Registros com coordenadas de GPS zeradas (inconsistência de localização): {len(coordenadas_invalidas)}")

else: print(“⚠️ Erro: Algumas das colunas numéricas necessárias para a validação estão ausentes.”) ```

📝 Conclusão da Equipe (Consistência):A análise da dimensão de Consistência Interna revelou que a base de dados consolidada da Polícia Rodoviária Federal (PRF) mantém perfeita harmonia em suas contagens de feridos, classificações textuais de gravidade e integridade de valores negativos. Contudo, o diagnóstico disparou um alerta crítico de alta volumetria na Regra 4, identificando 11.236 registros onde a soma das subcategorias de estado físico (mortos, feridos, ilesos e ignorados) diverge do total reportado no campo geral de pessoas. Essa desconexão representa uma inconsistência estrutural crônica e expressiva no preenchimento ou na agregação dos envolvidos na origem, razão pela qual esse comportamento será alvo de investigação prioritária nas próximas etapas do projeto para mapear a causa raiz e estruturar os devidos tratamentos e correções dentro do script de transformação (transform.py).

3. Dimensão: Acurácia

Objetivo: Verificar se os valores existentes fazem sentido, analisar a tipagem correta e detetar outliers.


import pandas as pd

# 3.1 Validação de Domínio (Correção do Bug Crítico de Caixa Alta/Baixa)
if 'uf' in df.columns:
    lista_ufs_validas = ['AC', 'AL', 'AM', 'AP', 'BA', 'CE', 'DF', 'ES', 'GO', 'MA',
                         'MG', 'MS', 'MT', 'PA', 'PB', 'PE', 'PI', 'PR', 'RJ', 'RN',
                         'RO', 'RR', 'RS', 'SC', 'SE', 'SP', 'TO']
    
    # Aplicamos .str.upper() e .str.strip() para garantir uma comparação justa
    uf_normalizada = df['uf'].astype(str).str.upper().str.strip()
    ufs_invalidas = df[~uf_normalizada.isin(lista_ufs_validas)]
    print(f"✅ Regra 1: Registros com UF fora do domínio nacional: {len(ufs_invalidas)}")
    if len(ufs_invalidas) > 0:
        print(f"   ⚠️ UFs estranhas encontradas: {ufs_invalidas['uf'].unique()}")

# 3.3 Análise Avançada de Outliers Relativos (Proporções entre Vítimas)

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

print("📈 Regra 3: Análise Estatística de Limites e Proporções (Método IQR):\n")

# --- PARTE A: Outliers Isolados (Pessoas e Veículos) ---
metricas_gerais = ['pessoas', 'veiculos']
metricas_existentes = [c for c in metricas_gerais if c in df.columns]

# --- PARTE B: Outliers de Proporção Relativa por Acidente ---
# Criamos um DataFrame auxiliar focado apenas em acidentes com alguma vítima para as razões
df_vitimas = df[(df['mortos'] > 0) | (df['feridos_graves'] > 0) | (df['feridos_leves'] > 0)].copy()

if not df_vitimas.empty:
    df_vitimas['razao_mortos_leves'] = np.where(df_vitimas['feridos_leves'] > 0, df_vitimas['mortos'] / df_vitimas['feridos_leves'], np.nan)
    df_vitimas['razao_mortos_graves'] = np.where(df_vitimas['feridos_graves'] > 0, df_vitimas['mortos'] / df_vitimas['feridos_graves'], np.nan)
    df_vitimas['razao_graves_leves'] = np.where(df_vitimas['feridos_leves'] > 0, df_vitimas['feridos_graves'] / df_vitimas['feridos_leves'], np.nan)

colunas_proporcao = ['razao_mortos_leves', 'razao_mortos_graves', 'razao_graves_leves']
titulos_proporcao = ['Mortos / Feridos Leves', 'Mortos / Feridos Graves', 'Feridos Graves / Feridos Leves']

# --- CONFIGURAÇÃO DO PLOT UNIFICADO (5 Gráficos em Linha) ---
fig, axes = plt.subplots(1, 5, figsize=(22, 5))

# 1. Plotando as Métricas Gerais (Escala Isolada)
for i, col in enumerate(metricas_existentes):
    Q1 = df[col].quantile(0.25)
    Q3 = df[col].quantile(0.75)
    IQR = Q3 - Q1
    limite_superior = Q3 + 3.0 * IQR
    outliers_extremos = df[df[col] > limite_superior]
    
    print(f"   - Volumetria '{col}': Limite lógico até {int(limite_superior)}. Outliers: {len(outliers_extremos)} registros.")
    
    sns.boxplot(y=df[col], ax=axes[i], color='#4C72B0')
    axes[i].set_title(f'Outliers em:\n{col.capitalize()}')
    axes[i].set_ylabel('')

# 2. Plotando os Ratios de Proporção Relativa
for j, col in enumerate(colunas_proporcao):
    idx_ax = len(metricas_existentes) + j
    dados_validos = df_vitimas[col].dropna()
    
    if not dados_validos.empty:
        Q1 = dados_validos.quantile(0.25)
        Q3 = dados_validos.quantile(0.75)
        IQR = Q3 - Q1
        limite_sup_prop = Q3 + 3.0 * IQR
        outliers_proporcao = df_vitimas[df_vitimas[col] > limite_sup_prop]
        
        print(f"   - Razão '{titulos_proporcao[j]}': Limite de proporção até {limite_sup_prop:.2f}. Outliers: {len(outliers_proporcao)} registros.")
        
        sns.boxplot(y=df_vitimas[col], ax=axes[idx_ax], color='#E15759')
        axes[idx_ax].set_title(f'Proporção:\n{titulos_proporcao[j]}')
        axes[idx_ax].set_ylabel('')
    else:
        axes[idx_ax].text(0.5, 0.5, 'Sem dados', ha='center', va='center')

plt.tight_layout()
plt.show()

📝 Conclusão da Equipe (Acurácia): A análise da dimensão de Acurácia confirmou a conformidade dos dados consolidados da Polícia Rodoviária Federal (PRF), com a validação de domínio para uf indicando zero não-conformidades e a cardinalidade categórica atestando a padronização das variáveis textuais. No ecossistema numérico, o diagnóstico estruturado por Intervalo Interquartil (\(3 \times IQR\)) expandiu o mapeamento de anomalias sob duas perspectivas complementares: a análise univariada isolou os limites típicos de magnitude dos sinistros (identificando volumes de cauda longa para ocorrências acima de 5 pessoas e 5 veículos), enquanto a análise bivariada cruzou as dinâmicas de letalidade e severidade física por evento. O cálculo estatístico das razões de proporcionalidade interna (Mortos / Feridos e Graves / Leves) permitiu delimitar o comportamento esperado de fragmentação das forças de impacto, isolando os acidentes cujas taxas de letalidade relativa destoam da distribuição histórica e qualificando a base como acurada e livre de ruídos espúrios induzidos por erros de digitação.

4. Dimensão: Temporalidade

Objetivo: Avaliar a correção das datas e a existência de lacunas no tempo ou datas futuras.

import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

print("📅 Iniciando Testes de Temporalidade...\n")

if 'data_inversa' in df.columns:
    # 1. Parsing Defensivo: Suporta múltiplos formatos automaticamente (ISO e BR)
    df['data_validada'] = pd.to_datetime(df['data_inversa'], errors='coerce', format='mixed')
    
    datas_invalidas = df['data_validada'].isnull().sum()
    print(f"❌ Regra 1: Registros com datas corrompidas ou inválidas: {datas_invalidas}")
    
    if datas_invalidas < len(df):
        data_min = df['data_validada'].min()
        data_max = df['data_validada'].max()
        print(f"⏱️  Regra 2: Cobertura da Série Histórica:")
        print(f"   - Ponto inicial dos dados: {data_min.strftime('%d/%m/%Y')}")
        print(f"   - Ponto final dos dados: {data_max.strftime('%d/%m/%Y')}")
        
        # 2. Check de Viagem no Tempo: Registros com datas além do dia de hoje
        data_limite = pd.Timestamp.now()
        futuro = df[df['data_validada'] > data_limite]
        print(f"❌ Regra 3: Registros com datas futuras aberrantes: {len(futuro)}")
        if len(futuro) > 0:
            print(f"   ⚠️ Amostra de datas futuras: {futuro['data_inversa'].unique()[:5]}")
            
        # 3. Investigação de Lacunas (Gaps Críticos):
        # Agrupa por dia para ver se há algum dia inteiro sem NENHUM acidente registrado no Brasil
        contagem_diaria = df['data_validada'].value_counts().sort_index()
        intervalo_completo = pd.date_range(start=data_min, end=data_max, freq='D')
        dias_zerados = len(intervalo_completo) - len(contagem_diaria)
        print(f"❌ Regra 4: Dias completos sem nenhum registro na série (Lacunas): {dias_zerados}")

        # 4. Gráfico de Tendência e Sazonalidade (Substituindo as barras poluídas por Linha Temporal)
        df_agrupado = df['data_validada'].dt.to_period('M').value_counts().sort_index()
        
        plt.figure(figsize=(12, 4.5))
        df_agrupado.plot(kind='line', marker='o', color='#2b5c8f', linewidth=2, markersize=4)
        plt.title('Evolução Mensal do Volume de Acidentes (Série Histórica)', fontsize=12, fontweight='bold')
        plt.xlabel('Linha do Tempo (Ano-Mês)')
        plt.ylabel('Quantidade de Ocorrências')
        plt.grid(True, linestyle='--', alpha=0.6)
        plt.tight_layout()
        plt.show()
        
    # Limpeza da coluna temporária do notebook
    df.drop(columns=['data_validada'], errors='ignore', inplace=True)
else:
    print("⚠️ Erro: A coluna 'data_inversa' não foi localizada no DataFrame.")

📝 Conclusão da Equipe (Temporalidade): A análise da dimensão de Temporalidade constatou a consistência cronológica e a perfeita sincronização dos dados integrados da Polícia Rodoviária Federal (PRF). O procedimento de parsing defensivo indicou zero registros com formatos inválidos ou corrompidos, delimitando uma série histórica contínua e sem quebras de continuidade. A validação de limites temporais confirmou a inexistência de anomalias relacionadas a datas futuras, enquanto o teste de descontinuidade diária acusou zero lacunas (gaps) na série, comprovando que não existem dias sem monitoramento operacional ao longo do horizonte temporal. Por fim, o mapeamento evolutivo mensal revelou uma distribuição de dados que flutua de forma controlada e previsível, capturando ciclos sazonais legítimos da malha rodoviária federal e qualificando a base como atualizada e íntegra para análises de tendências longitudinais.

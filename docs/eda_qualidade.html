Introdução

Este notebook tem como objetivo documentar o perfil de qualidade dos dados brutos de acidentes da PRF (camada raw/). O diagnóstico guiará as decisões de tratamento que serão implementadas no script transform.py. A análise está dividida nas quatro dimensões de Batini et al. (2009): Completude, Consistência, Acurácia e Temporalidade.

4. Dimensão: Temporalidade

Objetivo: Avaliar a correção das datas e a existência de lacunas no tempo ou datas futuras.

if 'data_inversa' in df.columns:
    # Tenta converter para datetime para validar
    df['data_parsed'] = pd.to_datetime(df['data_inversa'], errors='coerce', dayfirst=True)

    datas_invalidas = df['data_parsed'].isnull().sum()
    print(f"Datas com formato inválido/incompreensível: {datas_invalidas}")

    if datas_invalidas < len(df):
        print(f"Data mais antiga: {df['data_parsed'].min()}")
        print(f"Data mais recente: {df['data_parsed'].max()}")

        # Tem dados no futuro?
        futuro = df[df['data_parsed'] > pd.Timestamp.now()]
        print(f"Registos com data no futuro: {len(futuro)}")

        # Histograma para ver a distribuição ao longo do tempo
        plt.figure(figsize=(10, 4))
        df['data_parsed'].dt.to_period('M').value_counts().sort_index().plot(kind='bar')
        plt.title('Volume de Acidentes por Mês')
        plt.xlabel('Ano-Mês')
        plt.ylabel('Qtd Registos')
        plt.show()
